Roadmap de Aprendizado de Git

Roadmap de Aprendizado de Git

Fase 1: Introdução (1-3 meses)

  • Instalação e configuração
  • Comandos básicos (init, add, commit, log)
  • Repositórios locais e remotos

Fase 2: Fundamentos (3-6 meses)

  • Branches (branch, checkout, merge)
  • Merge e conflitos
  • Git Remote (push, pull, fetch)

Fase 3: Avançado (6-12 meses)

  • Git Rebase
  • Git Cherry-pick
  • Git Submodules
